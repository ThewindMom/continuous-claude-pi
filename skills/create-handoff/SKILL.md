# Create Handoff

Use the `cc_create_handoff` tool to write a handoff for the current session.

After creating it:
- inspect the generated path
- summarize what the next session should do first
- tell the user where it was saved

If the current session involves autonomous or research artifacts, mention the most important related paths in your response.
